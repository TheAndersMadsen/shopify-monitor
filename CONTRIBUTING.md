# Contributing to Shopify Monitor

Thank you for your interest in contributing to Shopify Monitor!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/yourusername/shopify-monitor.git`
3. Create a branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Test your changes
6. Commit your changes: `git commit -m "Add your commit message"`
7. Push to your fork: `git push origin feature/your-feature-name`
8. Open a Pull Request

## Development Setup

```bash
# Install dependencies
bun install

# Run the server
bun run dev
```

## Code Style

- Use TypeScript for all new code
- Follow existing code style and patterns
- Keep functions focused and single-purpose
- Add comments only when necessary for complex logic

## Testing

Before submitting a PR, please ensure:
- The server starts without errors
- The dashboard loads correctly
- WebSocket connections work
- Configuration can be saved and loaded

## Pull Request Process

1. Update the README if needed
2. Update version numbers if applicable
3. Ensure all changes are tested
4. Write clear commit messages
5. Reference any related issues

Thank you for contributing!
